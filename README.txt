ReactJS New
Lệnh tạo các thư viện trong reactJs
- Tạo project: yarn create react-app project_name
- Validate formik yup: yarn add formik yup
- Tạo axios gọi API: yarn add axios
- Dùng helmet tạo title cho từng component: yarn add react-helmet
- Lệnh thông báo: yarn add react-toastify
- Cài thư viện confirm dialog: yarn add react-bootstrap bootstrap
https://react-bootstrap.netlify.app/docs/components/modal/
